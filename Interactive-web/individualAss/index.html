Introduction

This assignment is composed of 50 hands-on practical exercises for students to practice JavaScript, covering objects, arrays, strings, HTML elements, and other key elements. These exercises will provide a comprehensive introduction to various JavaScript concepts, helping students improve their coding skills.

1. Create an array of 20 numbers and write a function to find the sum of all numbers.

2. A function to find the largest number in an array of 20 numbers.

3. A function to sort an array of 20 numbers in ascending order.

4. A function to reverse an array of 20 numbers.

5. A function that takes an array of 20 numbers and returns only even numbers.

6. A function that takes an array of 20 numbers and returns only odd numbers.

7. A function that takes an array of 20 numbers and returns the average of those numbers.

8. A function to remove duplicates from an array of 20 numbers.

9. A function that takes an array of 20 numbers and returns the median value.

10. A function to multiply all elements in an array of 20 numbers by 2.

11. Create an array of 20 names and write a function to print each name on a new line.

12. A function to sort an array of 20 names alphabetically.

13. A function that checks if a specific name exists in an array of 20 names and returns true or false.

14. A function to find the longest name in an array of 20 names.

15. A function to find names that start with a specific letter in an array of 20 names.

16. A function to convert all names to uppercase in an array of 20 names.

17. A function to remove a specific name from an array of 20 names.

18. A function to count how many names contain the letter "a" in an array of 20 names.

19. A function to concatenate all names in an array of 20 names into a single string, separated by commas.

20. A function to reverse the order of an array of 20 names.

21. Create a JavaScript object that represents a person (with properties: name, age, and occupation).

22. A function that takes a person object and returns a formatted string with their information.

23. A function to add a new property to an existing object.

24. A function to remove a property from an existing object.

25. A function to list all properties and values of a JavaScript object.

26. A function to merge two objects into one.

27. A function to check if a property exists in an object.

28. A function to clone an object without referencing the original one.

29. A function that loops through an array of objects and prints a specific property of each object.

30. A function that updates a specific property of an object based on user input.

31. A function that takes a string and reverses the string.

32. A function to count the number of vowels in a given string.

33. A function to convert a string to uppercase without using built-in methods.

34. A function to find the length of a string without using .length.

35. A function to replace all spaces with hyphens in a string.

36. A function to extract the first 10 characters from a string.

37. A function to find and replace a word in a string.

38. A function to split a string into an array of words.

39. A function to remove all numbers from a string.

40. A function to insert another string in the existing starting at position 7.

41. A JavaScript function that changes the background color of a webpage when a button is clicked.

42. A JavaScript function to hide an HTML element when a button is clicked.

43. A JavaScript function to create a new paragraph element and append it to the body.

44. A JavaScript function to remove all list items from an unordered list.

45. A JavaScript function to change the text content of a specific HTML element.

46. A JavaScript function to toggle a CSS class on an HTML element.

47. A JavaScript function to count the number of divs on a webpage.

48. A JavaScript function to get the value of a text input field and display it in a paragraph.

49. A JavaScript function to create a table with 3 rows and 3 columns, and populate it with numbers of your choice.

50. A JavaScript function to validate a form (check if name, email, and password fields are filled).
